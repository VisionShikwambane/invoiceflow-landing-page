# Invoiceserve Landing Page

Static landing page for **Invoiceserve** — a simple invoicing tool built for South African freelancers, contractors, and small businesses.

## Quick Start

No build step required. Just open `index.html` in your browser:

```bash
# Option 1: Open directly
open index.html

# Option 2: Use any static server
npx serve .
```

## Project Structure

```
├── index.html    # Landing page markup
├── styles.css    # All styles (layout, components, responsive)
├── main.js       # FAQ toggle, dynamic year, toast notifications
└── README.md
```

## Deploy

This is a static site — deploy anywhere:

- **GitHub Pages** — push to `main`, enable Pages in repo settings
- **Netlify** — drag and drop the folder
- **Vercel** — import the repo
- **Cloudflare Pages** — connect the repo
